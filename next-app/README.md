# YieldMap — Next.js Production App

This is the production Next.js version of YieldMap. The prototype is in the repo root (`src/App.jsx`).

## Setup

1. Copy `.env.local.example` to `.env.local` and fill in your API keys
2. `npm install`
3. `npm run dev`
4. Visit `localhost:3000/test` to validate API connectivity

## Deployment

Deploy to Vercel with root directory set to `next-app/`.
